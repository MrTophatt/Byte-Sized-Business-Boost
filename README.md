# Byte-Sized Business Boost

A web tool to help users discover and support small, local businesses in their community. This project includes user authentication, business browsing, favoriting, reviewing, and smooth UI interactions.

## Features

- User login with Google Sign-In
- View local businesses in a clean, interactive main page
- Favorite and review businesses
- Filter and sort businesses by category, rating, or name
- Search businesses quickly
- Responsive and animated UI for a polished user experience

## Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Database: MongoDB
- Authentication: Google OAuth
- Version control: Git + GitHub
